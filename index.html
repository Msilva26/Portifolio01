  • HTML

    O HTML é fundamental, pois é a base de qualquer página web. Com um bom entendimento de HTML, você pode criar estruturas bem organizadas e semânticas

  • CSS

    usado para estilizar a aparência de uma página web. Ele permite controlar o layout, as cores, as fontes e outros aspectos visuais de elementos HTML.

    O JS permite adicionar interatividade e comportamento dinâmico a uma página web. Ele pode ser usado para criar animações, manipular dados, validar formulários e muito mais
